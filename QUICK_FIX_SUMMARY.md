## 🐛 故障排除

如果记忆仍然不显示：

1. **检查 API 返回**
   ```
   按 F12 → Console
   查看 "🔍 [DEBUG]" 日志
   ```

2. **检查数据存储**
   ```
   按 F12 → Application → LocalStorage
   查看 mem_long_term 内容
   ```

3. **检查浏览器支持**
   ```
   localStorage 是否可用？
   HTML 注释是否被支持？
   ```

4. **重新加载**
   ```
   Ctrl+Shift+R（硬刷新）
   清除缓存后重试
   ```
